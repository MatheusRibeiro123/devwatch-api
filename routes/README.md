# 🧭 DevWatch

API de monitoramento de sistema desenvolvida com **FastAPI**, com o objetivo de coletar informações em tempo real sobre o uso de recursos da máquina.

## 🚧 Status do Projeto

Este projeto ainda está em desenvolvimento.

Está sendo utilizado como prática para aprendizado de **FastAPI**, organização de código backend e construção de APIs com propósito real.

## 🎯 Objetivo

Desenvolver uma API capaz de:

* Monitorar uso de CPU
* Monitorar uso de memória RAM
* Monitorar uso de disco
* Evoluir futuramente para histórico de dados e geração de alertas

## ⚙️ Tecnologias utilizadas

* Python
* FastAPI
* Uvicorn
* psutil

## 🧠 Aprendizados

Este projeto está sendo desenvolvido com foco em:

* Estruturação de APIs com FastAPI
* Separação de responsabilidades (routes/services)
* Organização de projetos backend
* Coleta de dados do sistema com psutil
